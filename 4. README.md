# OumarouTracker

Application web de suivi de candidatures d’alternance.

## 🎯 Objectif
Organiser et suivre mes candidatures d’alternance.

## ⚙️ Fonctionnalités
- Ajout de candidatures
- Modification des statuts
- Suppression
- Recherche
- Sauvegarde locale

## 💻 Technologies
- HTML
- CSS
- JavaScript

## 🚀 Lancer le projet
Ouvrir index.html dans un navigateur.
